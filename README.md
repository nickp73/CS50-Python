# CS50 Programming with Python

A collection of programming assignments from Harvard's CS50 Introduction to Computer Science course, focusing on Python fundamentals.

## Overview

This repository contains solutions to various CS50 Python problem sets, covering core programming concepts including:
- Variables, data types, and basic operations
- Control flow and functions
- Object-oriented programming
- Libraries and modules

## Project Structure

### Assignments Included

- **bank.py** - Currency conversion and banking operations
- **compare.py** - String comparison and conditional logic
- **deep.py** - Recursive depth-first exploration
- **einstein.py** - Physics calculation (E=mc²)
- **extensions.py** - File extension validation
- **faces.py** - Text processing and pattern recognition
- **grade.py** - Grade calculation and evaluation
- **house.py** - Conditional output generation
- **interpreter.py** - Expression evaluation and arithmetic
- **interpreter_one.py** - Alternate interpreter implementation
- **parity.py** - Even/odd number checking
- **playback.py** - Speed-adjusted playback simulation
- **tip.py** - Tip calculator with percentage logic

## Requirements

- Python 3.8 or higher

## Running the Programs

Each Python file is a standalone program. Run any assignment with:

```bash
python3 filename.py
```

Some programs may require command-line arguments or user input—refer to each file's docstring for specific instructions.
